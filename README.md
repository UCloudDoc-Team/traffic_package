
# 概览
共享流量包UTP（UCloud Traffic Package）是预付费流量套餐产品，包含全时流量包和闲时流量包。流量包购买后立即生效，并自动抵扣对应地域按流量计费IP产生的公网流量费用。相比于流量后付费，共享流量包具有易于管理、单价更低等优势。
